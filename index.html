DSA VLOP Transparency Query API

transparency.kieranmaynard.com · live

A live research API and interactive dashboard over EU Digital Services Act transparency data from 33 designated Very Large Online Platforms (H2 2025). An async job / poll API with a safe, structured query interface modelled on the TikTok Research API — no arbitrary SQL accepted. Browse cross-platform enforcement statistics, run queries across 9 DSA report tables, or ask questions in plain English using the Claude-powered NL Ask box. Open-source, deployed on Cloud Run.
