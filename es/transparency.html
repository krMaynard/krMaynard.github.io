Análisis

Datos de transparencia

Paneles interactivos construidos a partir de informes de transparencia de plataformas — combinando datos que las empresas publican por separado entre servicios o periodos en una única vista comparativa. Todo el procesamiento ocurre en tu navegador; no se envía ningún dato a un backend.

Destacado · API en vivo

API de consulta de transparencia VLOP del DSA

Una API de investigación en vivo y un panel interactivo sobre los datos de transparencia de la Ley de Servicios Digitales de la UE de 33 plataformas en línea de muy gran tamaño designadas (H2 2025). Una API de trabajos asíncronos con sondeo y una interfaz de consulta estructurada y segura basada en la Research API de TikTok. Explora estadísticas de aplicación entre plataformas, ejecuta consultas estructuradas sobre 9 tablas de informes del DSA o formula preguntas en lenguaje natural con el cuadro «Ask» en inglés impulsado por Claude.

FastAPI EU DSA API de consulta estructurada Claude AI Cloud Run Proyecto personal

App · Android y escritorio

Transparency — Cliente móvil y de escritorio

Una app nativa de Kotlin Multiplatform para la API de consulta de transparencia. Explora estadísticas de aplicación entre plataformas, solicitudes de retirada gubernamentales y ejecuta consultas estructuradas desde tu teléfono o escritorio — los mismos datos en vivo que el panel web, en una interfaz Material 3 pensada primero para el móvil y localizada en ocho idiomas.
