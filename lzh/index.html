Kieran Maynard（齊冉）

人工智能產品之經理暨建造者

吾謀而手築能任實事之人工智能器——大模型之渠、自主之智能體、研究之 API——致之於生產之境。歷谷歌、Roblox 十有一年，廣施人工智能於信安、透明、合規。

吾本出於純人文之門——語言之學、東亞之文、翻譯之術——以好奇與所迫，自學編程。初未嘗志為產品之經理；然循路而築：自動之腳本、SQL 之渠、RAG 暨自主智能體之系、Chrome 之擴。所築既足，乃證吾本即經理也。

吾之所能

人工智能暨大語言模型之器

RAG 之問答機、巨量推理之渠、自主之智能體，及為企業之大運營而設之人工智能自動化。

施於信安暨透明之人工智能

施人工智能與自動化於透明之報、內容之審、律令之責（歐盟 DSA、CMA、AB 587）——化信安之務為可擴可稽之系。

數據暨自動化

精深之 SQL、Python 之渠、終始一貫之自動化——以可信可複之系，代高危之手工。

跨職之統御

化繁難之監管與商務之需，為技術之向，貫工程、法務、政策、暨上位諸君。

代表之器

DSA VLOP 透明度查詢 API

transparency.kieranmaynard.com · 現行

據歐盟數字服務法（DSA）所指三十有三超大在線平台（二〇二五下半）之透明數據，立一現行之研究 API 及交互之儀錶板。仿 TikTok 研究 API 而設，異步任務、輪詢以求；查詢之介安而有度，不納任意之 SQL。可覽跨平台之執法統計，可於九張 DSA 報表之上運其查詢，亦可借 Claude 所驅之自然語問框、以白話相詢。其源公開，部署於 Cloud Run。
