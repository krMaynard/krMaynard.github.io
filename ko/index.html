키어런 메이나드

AI·데이터 프로덕트 매니저

복잡한 규제·컴플라이언스 과제를 확장 가능한 자동화 시스템으로 전환하는 AI 기반 도구를 만듭니다. Google과 Roblox에서 11년 이상 프로덕트 매니지먼트·컴플라이언스·Trust & Safety 분야에 종사했습니다.

저는 순수 인문학—언어학, 동아시아 문학, 번역—을 전공했으며, 호기심과 필요에 의해 독학으로 코딩을 익혔습니다. 처음부터 PM을 목표로 한 것은 아닙니다. 자동화 스크립트, SQL 파이프라인, AI 시스템, Chrome 확장 프로그램을 하나씩 만들다 보니 어느새 PM이 되어 있었습니다.

전문 분야

AI·LLM 제품

RAG 챗봇, 대규모 LLM 추론 파이프라인, 에이전틱 시스템, 엔터프라이즈 운영을 위한 AI 기반 자동화.

규제·컴플라이언스 PM

EU DSA, CMA 약정, 콘텐츠 모더레이션 정책. 법적 의무를 확장 가능하고 감사 대응이 가능한 시스템으로 구현.

데이터·자동화

고급 SQL, Python 파이프라인, 엔드투엔드 자동화 — 고위험 수동 프로세스를 신뢰할 수 있고 반복 가능한 시스템으로 대체.

부서 간 협업 리더십

복잡한 규제 및 비즈니스 요구사항을 엔지니어링·법무·정책·경영진 등 다양한 이해관계자가 따를 수 있는 기술적 방향으로 변환.

주요 프로젝트

선택된 프로젝트

Roblox Research Data API

Roblox 최초의 외부 연구자용 공개 API 설계를 주도하고 있습니다. 학계, 시민사회 단체, 플랫폼 안전 연구자들이 Roblox의 공개 데이터를 프로그래밍 방식으로 조회할 수 있도록 지원하여 플랫폼 투명성을 새로운 차원으로 끌어올립니다.

API Design Research Access Platform Transparency Roblox

Sandpiper — RAG AI 어시스턴트

900명 규모의 Privacy Sandbox 워킹 그룹을 위한 엔드투엔드 RAG 지식 어시스턴트를 설계했습니다. 12,000건 이상의 접근 제한 문서를 인덱싱했으며, Google Privacy Sandbox TPgM AI 해커톤에서 2위를 수상했습니다.

RAG Gemini LLM System Design Google

LLM 사건 요약 대시보드

SQLMiner를 사용해 수천 건의 법적 삭제 요청을 자동 요약하는 대규모 LLM 추론 파이프라인을 구축했습니다. AI가 생성한 요약이 팀의 주요 관리 대시보드에 그대로 노출됩니다.

Bulk Inference SQL Prompt Engineering Google

Chrome Extension node.js Browser Automation Google

컴플라이언스 보고 파이프라인

CMA 위임 분기별 보고서를 엔드투엔드 ETL 파이프라인으로 자동화했습니다. 생성 시간을 60~80 인시에서 5시간 미만으로 단축(10배 감소), 오류율 0%, 감사 추적성 >95% 달성. "역대 가장 순조로운 분기였습니다." — 법무팀
