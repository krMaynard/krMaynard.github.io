컨설팅 서비스

프로덕트 매니지먼트, 프로세스 자동화, 언어 습득 분야의 컨설팅을 제공합니다. PM 컨설팅은 Google과 Roblox에서 11년 이상의 직접 경험에서 비롯된 것으로, 이론이 아닙니다. 자동화와 언어 분야는 엔터프라이즈 규모의 파이프라인 구축과 수년간의 번역 실무에서 쌓은 동일한 엄밀함으로 접근합니다.

프로젝트를 논의하려면 kieranmaynard{at}gmail{dot}com으로 연락주세요.

프로덕트 매니지먼트

문제 정의에서 로드맵, 스펙, 실행까지 전체 PM 라이프사이클을 지원합니다. 특히 다음 경우에 유용합니다:

  • 초기 단계 제품: 범위 정의, PRD 작성, 출시 기준 구조화
  • 규제 및 컴플라이언스 PM: 법적 의무(DSA, CMA, AB 587 등)를 제품 요구사항으로 전환
  • 이해관계자 정렬: 경영진 보고서, 크로스펑셔널 의사결정 문서, 에스컬레이션 프레임워크 준비
  • PM으로의 커리어 전환: 포트폴리오 검토, 이력서 코칭, 면접 준비

자동화 및 데이터

고위험 수동 프로세스를 신뢰할 수 있고 반복 가능한 시스템으로 대체하는 자동화를 설계하고 구축합니다. 과거 작업에는 60–80 인시 컴플라이언스 보고 사이클을 5시간 미만으로 단축한 것, 수만 건의 케이스를 처리한 Chrome 확장 프로그램으로 $50k+ 벤더 비용을 절감한 것이 포함됩니다.

  • SQL & ETL 파이프라인: BigQuery, Hive, Spark — 데이터 정제, 변환, 운영화
  • Python 자동화: 스크립팅, pandas, 작업 파이프라인
  • 브라우저 자동화: Chrome 확장 프로그램, node.js 도구
  • LLM 통합: 프롬프트 엔지니어링, 대규모 추론 파이프라인, RAG 시스템 설계
  • 프로세스 감사: 수동 병목 지점 파악 및 자동화 대안 설계

언어 습득

일본어, 중국어(보통화 및 광동어), 한국어를 전문가 수준으로 습득했으며, 수년간 다른 학습자들이 체계적이고 효율적인 언어 학습 방법을 구축할 수 있도록 도왔습니다. 시험 준비가 아닌 실용적인 유창성—읽기, 말하기, 전문적 사용—에 집중합니다.

  • 일본어: 모든 레벨, 특히 문학, 법률, 기술 분야에서의 깊이
  • 중국어(보통화): 일상 회화부터 전문 수준까지; 고전 중국어 요청 가능
  • 한국어: 일상 회화 및 문어; 읽기 및 듣기 중점
  • 학습 시스템 설계: SRS 설정, 자료 선택, 읽기 단계, 이머전 전략
  • 전문적 언어 사용: 이메일, 비즈니스 커뮤니케이션, 목표 언어로 된 문서 읽기

번역 작업 예시: 무라카미 하루키(일본어), 김언수(한국어), 장자가(중국어).
