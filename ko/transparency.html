분석

투명성 데이터

플랫폼 투명성 보고서를 기반으로 구축된 인터랙티브 대시보드 — 기업들이 서비스나 기간별로 별도로 공개하는 데이터를 단일 비교 뷰로 결합합니다. 모든 처리는 브라우저에서 이루어지며 데이터는 백엔드로 전송되지 않습니다.

주요 프로젝트 · 실시간 API

DSA VLOP 투명성 쿼리 API

33개 지정 초대형 온라인 플랫폼(H2 2025)의 EU 디지털 서비스법 투명성 데이터를 제공하는 실시간 연구용 API 및 인터랙티브 대시보드. TikTok Research API를 모델로 한 안전한 구조화 쿼리 인터페이스를 채용했으며, 임의 SQL은 허용하지 않습니다. 크로스플랫폼 집행 통계를 탐색하고, 9개 DSA 보고서 테이블에 대한 구조화 쿼리를 실행하거나, Claude 기반 자연어 질문 기능을 사용하세요.
