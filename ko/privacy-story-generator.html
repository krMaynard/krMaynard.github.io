개인정보처리방침 — 스토리 제너레이터

최종 업데이트: 2026년 6월

본 개인정보처리방침은 Google Gemini를 사용해 초기 독자를 위한 맞춤형 해독용(decodable) 도서를 생성하는 개인용 도구인 Android 앱 Story Generator(이하 “본 앱”)에 적용됩니다.

기기에 저장되는 데이터

본 앱은 다음 데이터를 사용자의 기기에만 로컬로 저장합니다:

  • Gemini API 키 — flutter_secure_storage를 통해 기기의 보안 키스토어에 저장됩니다. 사용자가 직접 시작한 Google Gemini API 요청을 제외하고는 기기를 벗어나지 않습니다.
  • 아동 프로필 — 이야기를 맞춤화하기 위해 입력하는 이름, 나이, 관심사, 읽기 수준. 로컬 앱 환경설정에 저장됩니다.
  • 이야기 사양 및 세션 상태 — 제목, 테마, 일견 단어(sight words) 등 입력한 항목. 로컬 앱 환경설정에 저장됩니다.
  • 생성된 이야기와 이미지 — 기기의 로컬 저장소에만 저장됩니다. 공유는 항상 사용자가 명시적으로 시작합니다.

이 데이터 중 어느 것도 아래에 설명된 Google을 제외하고는 본 앱 개발자나 제3자에게 전송되지 않습니다.

Google Gemini로 전송되는 데이터

생성하기를 누르면, 본 앱은 이야기 사양(테마, 일견 단어, 캐릭터 이름 등)과 관련 프로필 항목(아동의 이름, 나이, 읽기 수준)을 Google Gemini API로 전송하여 이야기 텍스트와 삽화를 생성합니다. 이 데이터는 Google 개인정보처리방침 및 Gemini API 서비스 약관의 적용을 받습니다. 본 앱은 Google이 이 데이터를 처리하거나 보관하는 방식을 제어하지 않습니다.

수집하지 않는 데이터

본 앱은 분석 데이터, 충돌 보고서, 기기 식별자, 사용 통계를 수집하지 않습니다. 광고 SDK도, 추적 픽셀도, 계정·로그인 요건도 없습니다.

아동의 개인정보

본 앱은 성인이 운영하는 개인용 육아 도구입니다. 아동을 대상으로 하지 않으며, 본 앱을 사용하는 성인과 별개로 아동으로부터 개인정보를 고의로 수집하지 않습니다. 부모가 입력한 아동 프로필 데이터는 부모의 기기에만 저장됩니다.
