Gemini Code Assist의 GitHub 리뷰어가 오늘 종료됩니다 — 다음은 어디로

2026년 7월 17일
Gemini Code Assist AI 코드 리뷰 개발자 도구 GitHub

Gemini에게 풀 리퀘스트 리뷰를 맡겨 왔다면, 오늘이 바로 그 봇이 조용해지는 날입니다. 2026년 7월 17일부로 구글은 GitHub용 소비자 버전 Gemini Code Assist를 종료했습니다 — 여러분의 PR에 자동화된 코드 리뷰와 요약을 남겨 주던 그 앱 말입니다. 봄부터 단계적으로 진행되어 온 무료·개인 등급 축소의 마지막 절차입니다.

각 요소가 서로 다른 날짜에 꺼졌으므로, 전체 순서를 짚어 보겠습니다.

  • 2026년 6월 18일 — Gemini Code Assist IDE 확장 프로그램과 Gemini CLI가 개인용 Gemini Code Assist 및 Google AI Pro·Ultra 요금제에 대해 요청 처리를 중단했습니다.
  • 2026년 7월 17일 (오늘) — 소비자용 Gemini Code Assist GitHub 앱이 코드 리뷰 활동을 종료합니다.

누구도 성급하게 걱정하기 전에 짚어 둘 중요한 단서 하나 — 이것은 소비자 이야기입니다. 여러분의 조직이 Gemini Code Assist Standard나 Enterprise 라이선스를 유료로 이용하고 있다면, 구글은 그것들이 영향을 받지 않는다고 밝힙니다 — IDE 어시스트, CLI, GitHub 리뷰가 그대로 유지됩니다. 끝나는 것은 많은 1인 개발자와 소규모 팀이 자신의 저장소에 연결해 두었던 무료·개인용 진입로입니다.

구글이 여러분에게 바라는 것

구글이 제시하는 마이그레이션 경로는 5월 I/O에서 공개한 "에이전트 우선" 플랫폼 Antigravity와, 그 터미널 동반 도구인 Antigravity CLI입니다 — Antigravity 2.0 데스크톱 앱과 동일한 에이전트 하니스를 공유하는 Go 재작성판입니다. 5월 19일부터 antigravity.google/download에서 이용할 수 있으며, 여러분의 Agent Skills, Hooks, Subagents, Extensions(이제는 "플러그인")가 그대로 이어집니다. 구글은 출시 시점에 완전한 기능 동등성은 없다고 솔직하게 밝히고 있습니다.

하지만 그 간극에 주목하세요. Antigravity CLI는 여러분이 조종하는 에이전트입니다 — 터미널과 에디터에서 동작하는 대화형 코딩 어시스턴트죠. 오늘 실제로 꺼진 것은 요청하지 않아도 여러분의 GitHub PR을 지켜보고 댓글을 달아 주던 자동 리뷰어였습니다. 이 둘은 같은 일이 아니며, 구글은 소비자용 GitHub 봇을 그대로 대체할 물건을 내놓고 있지 않습니다. 그러니 여러분이 아쉬워할 것이 바로 그 PR 리뷰라면, 솔직한 답은 이렇습니다. Antigravity만으로는 부족하고, 전용 코드 리뷰 도구가 필요할 가능성이 높습니다.

GitHub PR 리뷰 작업을 위한 대안

좋은 소식 — 지금 이 분야는 개발자 도구 시장에서 가장 붐비는 구석 중 하나입니다. 대략 여러분이 바꿔야 할 것이 적은 순서대로 정리하면 이렇습니다.

  • GitHub Copilot code review — 이미 Copilot을 유료로 쓰고 있다면 가장 저항이 적은 길입니다. GitHub에 기본 내장되어 있어 새로운 연동이 전혀 필요 없고, 요약 아래에 CodeQL과 ESLint 스캔을 얹어 줍니다. 전용 도구보다는 얕지만, 켜는 즉시 쓸 수 있습니다.
  • CodeRabbit — 가장 인기 있는 전용 AI 리뷰어로, PR 안에 구조화된 줄 단위 피드백을 바로 남깁니다. 유료 등급은 개발자당 월 $24 정도부터 시작하며, 오픈소스와 공개 저장소를 위한 무료 등급도 있습니다 — Gemini 봇이 하던 일에 가장 근접한 대체재입니다.
  • Qodo Merge(옛 Codium/PR-Agent) — PR 요약, 리뷰, 인라인 제안을 제공하며, 테스트 생성과 거버넌스까지 원하는 팀을 겨냥합니다. 핵심인 PR-Agent가 오픈소스라, 자신의 diff를 또 다른 SaaS에 넘기고 싶지 않다면 직접 셀프 호스팅할 수 있습니다.
  • Greptile — diff만이 아니라 코드베이스 전체의 깊은 맥락에 기댑니다. 더 넓은 저장소의 맥락에서만 드러나는 문제를 잡아내는 데 강합니다. 개발자당 월 $30 정도입니다.
  • Claude Code — 차라리 CI에서 에이전트를 돌리고 싶다면, 그 GitHub Action이 PR을 리뷰(그리고 PR을 열 수도)해 주므로 무엇을 검사할지 정확히 스크립트로 짤 수 있습니다. 호스팅형 봇보다 설정이 많지만, 워크플로를 여러분이 온전히 소유합니다. (밝혀 두자면, 이 글도 그것으로 초안을 작성했습니다.)

여러분의 충성심이 특별히 구글의 모델을 향해 있다면, Gemini는 이들 중 여럿을 통해 이용할 수 있다는 점에 유의하세요 — 봇을 포기하기 위해 모델까지 포기할 필요는 없습니다.

요점

구글이 AI 보조 개발을 죽인 것은 아닙니다. 통합한 것입니다. 소비자용 Gemini Code Assist 도구들은 Antigravity로 흡수되고 있고, 엔터프라이즈 라이선스는 변함없이 이어집니다. 하지만 우리 상당수가 기대어 온 그 특정한 편의 — 모든 PR에 조용히 댓글을 달아 주던 무료 리뷰어 — 는 오늘부로 사라졌고, Antigravity는 그것의 곧바른 대체품이 아닙니다. Antigravity로 옮겨 가면 구글의 세계 안에 머무르게 되고, Copilot의 리뷰어나 CodeRabbit, Qodo, Greptile을 집어 들면 PR 리뷰를 되찾게 됩니다. 어느 쪽이든, 다음 풀 리퀘스트가 리뷰 없이 지나가기 전에 대체재를 연결해 두기에 오늘이 좋은 날입니다.

출처: 구글의 Gemini Code Assist deprecations 페이지(종료 날짜와 엔터프라이즈 예외 조항)와 Gemini CLI → Antigravity CLI 전환 게시글, 그리고 9to5Google의 보도. 도구별 세부 사항은 2026년 7월 기준 각 벤더의 사이트에서 가져왔으며, 가격은 자주 바뀌므로 결정하기 전에 확인하세요.
