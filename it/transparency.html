Analisi

Dati di trasparenza

Dashboard interattive costruite a partire dai report di trasparenza delle piattaforme — combinano dati che le aziende pubblicano separatamente tra servizi o periodi in un'unica vista comparativa. Tutta l'elaborazione avviene nel tuo browser; nessun dato viene inviato a un backend.

In evidenza · API live

DSA VLOP Transparency Query API

Un'API di ricerca live e una dashboard interattiva sui dati di trasparenza del Digital Services Act dell'UE, provenienti da 33 Very Large Online Platform designate (H2 2025). Un'API asincrona job / poll con un'interfaccia di query strutturata e sicura, modellata sulla TikTok Research API. Esplora le statistiche di enforcement tra piattaforme, esegui query strutturate su 9 tabelle di report DSA o poni domande in linguaggio naturale usando il riquadro Ask basato su Claude.

FastAPI EU DSA Structured Query API Claude AI Cloud Run Progetto personale

App · Android & Desktop

Transparency — client mobile e desktop

Un'app nativa Kotlin Multiplatform per la Transparency Query API. Sfoglia le statistiche di enforcement multipiattaforma, le richieste governative di rimozione ed esegui query strutturate dal telefono o dal desktop — gli stessi dati live della dashboard web, in un'interfaccia Material 3 mobile-first e localizzata in otto lingue.
