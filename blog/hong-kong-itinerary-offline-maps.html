Mapping a Hong Kong Trip Offline with Claude Code

June 20, 2026
AI/LLM Engineering Travel Language

At the end of this month, six of us are going to Hong Kong — three generations, a stroller, and a shared bag of everyday medications. I wanted one document my Chinese-speaking family could open on their phones: a gentle four-day plan, written in plain Chinese, with a clear map for every stop. I handed the job to Claude Code in a single prompt. The writing was the easy part. The interesting part was that the machine had no way to reach a map.

The obvious approach, dead on arrival

This ran in a sandboxed cloud environment with a strict network egress allowlist — the kind of setup you actually want when you're letting an agent run loose in your repos. The first idea any of us would try is to drive a headless browser to Google Maps with the interface language set to Chinese and screenshot each location. It never got off the ground. Every map and tile service it reached for came back the same way: 403 — Host not in allowlist. Google Maps, OpenStreetMap's tile servers, Mapbox, Carto, Stadia, Wikimedia, even the free keyless vector basemaps. No tiles, no static-map API, nothing that draws a map.

The lazy move here is to stop and ask me to poke a hole in the firewall. It didn't.

What it did instead

It mapped the constraint first — probing what was reachable — and found that GitHub, npm, and PyPI were open. That turned out to be enough, because a map is just geometry plus a renderer, and both can come from a package registry.

So it pulled Natural Earth's 10-metre coastline — real vector geography, mirrored on GitHub — straight from the repo. It clipped that to a Hong Kong bounding box with shapely, then ran a sanity check I wouldn't have thought to write: does this data actually resolve Victoria Harbour? Concretely — do Kowloon and Hong Kong Island come back as two separate landmasses, or has the coarse coastline merged them into one blob? Two separate polygons came back. The harbour was there. Only then did it trust the data.

From there it became the map renderer. It projected the coastline to Web Mercator, drew water, land, and a few parks, dropped a teardrop pin and a Chinese label on every stop — the hotel in Tsim Sha Tsui, the Avenue of Stars, the Peak Tram, Ocean Park, the Space Museum, the airport out on Lantau — and added a scale bar, a north arrow, and a greedy label-placement pass so nothing collided. All of it as SVG, which it then rasterised to crisp PNGs using the one browser it did have locally: Playwright's Chromium, with a bundled CJK font rendering the Chinese. Ten maps, every one produced without a single network call at render time, wired into a mobile-friendly itinerary page that embeds them day by day.

The part that stuck with me

Not the rendering — the debugging loop. Probing a dozen hosts, reading the bodies of the 403s, noticing that raw.githubusercontent.com answered 200 while every map host refused, and then pivoting the whole strategy from "screenshot a map" to "fetch the geometry and be the map" — that is exactly the afternoon of fiddling a capable engineer would have spent, compressed into a few minutes. And the instinct to verify the coastline before drawing on it, rather than discovering a merged harbour in the final image, is the kind of care I want from a colleague, not just a code generator.

What I take from it

This rhymes with what I keep finding (in compliance work, in a security sweep): the wins land on structured tasks with verifiable outputs, and a hard constraint often improves the result instead of blocking it. A hand-rendered vector map with controlled Chinese typography is genuinely cleaner for my mother-in-law to read than a cluttered Google Maps screenshot would have been — bigger labels, fewer distractions, every name in the script she reads. The constraint didn't degrade the deliverable; it produced a better one.

We leave in ten days. The full plan — in Chinese, with all ten maps — lives on a single page the family can open on the plane. Hats, sunscreen, umbrella, and the medications are already on the list.
