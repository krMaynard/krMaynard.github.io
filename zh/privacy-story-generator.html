法律信息

隐私政策 — 故事生成器

最后更新：2026年6月

本隐私政策适用于 故事生成器（以下简称"本应用"）Android应用，这是一款利用 Google Gemini 为早期读者生成个性化图画书的个人工具。

存储在您设备上的数据

本应用仅在您的设备本地存储以下数据：

  • Gemini API 密钥 — 通过 flutter_secure_storage 存储在设备的安全密钥库中。除您主动发起的对 Google Gemini API 的请求外，该密钥不会离开您的设备。
  • 孩子的个人资料 — 您输入的用于个性化故事的姓名、年龄、兴趣爱好和阅读水平。存储在本地应用偏好设置中。
  • 故事规格及会话状态 — 您输入的标题、主题、常见词等内容。存储在本地应用偏好设置中。
  • 生成的故事和图像 — 仅保存到您设备的本地存储。分享操作始终需要您明确发起。

上述数据不会传输给本应用的开发者或除以下所述 Google 之外的任何第三方。

发送给 Google Gemini 的数据

当您点击"生成"时，本应用会将故事规格（主题、常见词、角色名称等）及相关个人资料字段（孩子的姓名、年龄、阅读水平）发送至 Google Gemini API 以生成故事文本和插图。这些数据受 Google 隐私政策 和 Gemini API 服务条款 约束。本应用无法控制 Google 处理或保留这些数据的方式。

我们不收集的数据

本应用不收集分析数据、崩溃报告、设备标识符或使用统计信息。应用中不包含任何广告 SDK、跟踪像素，也无需账户或登录。

儿童隐私

本应用是供成年人（家长）使用的个人育儿工具，不面向儿童，也不会独立收集儿童的个人信息。家长输入的儿童个人资料数据仅存储在家长的设备上。

政策变更

若本政策发生重大变更，更新后的版本将连同新的生效日期发布在此网址。
