Ausgewähltes Produkt

DSA VLOP Transparency Query API

transparency.kieranmaynard.com · live

Eine Live-Research-API und ein interaktives Dashboard über Transparenzdaten des EU Digital Services Act von 33 designierten Very Large Online Platforms (H2 2025). Eine asynchrone Job-/Poll-API mit einer sicheren, strukturierten Abfrageschnittstelle nach dem Vorbild der TikTok Research API — kein beliebiges SQL wird akzeptiert. Durchsuchen Sie plattformübergreifende Durchsetzungsstatistiken, führen Sie Abfragen über 9 DSA-Berichtstabellen aus oder stellen Sie Fragen in einfachem Englisch über die Claude-gestützte NL-„Ask"-Box. Open Source, bereitgestellt auf Cloud Run.
