Le relecteur GitHub de Gemini Code Assist s'éteint aujourd'hui — où aller ensuite

17 juillet 2026
Gemini Code Assist Revue de code par IA Outils pour développeurs GitHub

Si vous laissiez Gemini relire vos pull requests, c'est aujourd'hui que le bot se tait. Depuis le 17 juillet 2026, Google a désactivé la version grand public de Gemini Code Assist sur GitHub — l'application qui publiait des revues de code et des résumés automatisés sur vos PR. C'est la dernière étape d'un arrêt échelonné des offres gratuite et individuelle, en cours de déploiement depuis le printemps.

Voici la séquence complète, car les différents éléments se sont éteints à des dates distinctes :

  • 18 juin 2026 — les extensions IDE de Gemini Code Assist et la Gemini CLI ont cessé de traiter les requêtes pour Gemini Code Assist pour les particuliers ainsi que pour les forfaits Google AI Pro et Ultra.
  • 17 juillet 2026 (aujourd'hui) — l'application GitHub grand public Gemini Code Assist met fin à son activité de revue de code.

Une précaution importante avant que quiconque ne panique : il s'agit ici du volet grand public. Si votre organisation paie une licence Gemini Code Assist Standard ou Enterprise, Google indique que celles-ci sont non concernées — votre assistance dans l'IDE, la CLI et les revues GitHub continuent de fonctionner. Ce qui prend fin, c'est la porte d'entrée gratuite/individuelle que beaucoup de développeurs indépendants et de petites équipes avaient branchée sur leurs dépôts.

Ce que Google veut que vous fassiez

Le chemin de migration proposé par Google est Antigravity, la plateforme « agent-first » dévoilée à I/O en mai, accompagnée de son pendant en terminal, Antigravity CLI — une réécriture en Go qui partage le même harnais d'agent que l'application de bureau Antigravity 2.0. Elle est disponible depuis le 19 mai sur antigravity.google/download, et vos Agent Skills, Hooks, Subagents et Extensions (désormais « plugins ») sont repris. Google le dit franchement : il n'y a pas de parité fonctionnelle complète au lancement.

Mais notez l'écart. Antigravity CLI est un agent que vous pilotez — un assistant de codage interactif dans votre terminal et votre éditeur. Ce qui s'est réellement éteint aujourd'hui, c'était un relecteur automatisé qui surveillait vos PR GitHub et les commentait sans qu'on le lui demande. Ce ne sont pas les mêmes tâches, et Google ne livre pas de remplaçant équivalent pour le bot GitHub grand public. Donc si ce qui va vous manquer, ce sont les revues de PR, la réponse honnête est : il vous faut probablement un outil de revue de code dédié, et pas seulement Antigravity.

Alternatives pour la tâche de revue de PR sur GitHub

Bonne nouvelle — c'est l'un des recoins les plus encombrés du marché des outils pour développeurs en ce moment. À peu près par ordre de ce que vous auriez le moins à changer :

  • GitHub Copilot code review — la voie de moindre résistance si vous payez déjà pour Copilot. C'est natif à GitHub, cela ne requiert aucune nouvelle intégration, et cela superpose l'analyse CodeQL et ESLint sous les résumés. Plus superficiel que les outils dédiés, mais activable instantanément.
  • CodeRabbit — le relecteur IA dédié le plus populaire, qui publie des retours structurés ligne par ligne directement dans la PR. Les offres payantes démarrent autour de 24 $/développeur/mois, avec une offre gratuite pour l'open source et les dépôts publics — le remplaçant le plus proche de ce que faisait le bot Gemini.
  • Qodo Merge (anciennement Codium/PR-Agent) — résumés de PR, revue et suggestions en ligne, à destination des équipes qui veulent aussi de la génération de tests et de la gouvernance. Son cœur PR-Agent est open source, vous pouvez donc l'héberger vous-même si vous préférez ne pas confier vos diffs à un autre SaaS.
  • Greptile — s'appuie sur un contexte profond de l'ensemble de la base de code plutôt que sur le seul diff ; efficace pour repérer les problèmes qui n'ont de sens que dans le contexte du dépôt élargi. Autour de 30 $/développeur/mois.
  • Claude Code — si vous préférez faire tourner un agent en CI, sa GitHub Action relit les PR (et peut en ouvrir), vous scriptez donc exactement ce qui est vérifié. Plus de configuration qu'un bot hébergé, mais vous êtes maître du flux de travail. (En toute transparence : ce billet a été rédigé avec.)

Si votre fidélité va spécifiquement aux modèles de Google, notez que Gemini est disponible via plusieurs de ces outils — vous n'avez pas à abandonner le modèle pour abandonner le bot.

Ce qu'il faut en retenir

Google n'a pas tué le développement assisté par IA ; il l'a consolidé. Les outils grand public Gemini Code Assist sont intégrés à Antigravity, et les licences entreprise poursuivent leur cours sans changement. Mais la commodité précise sur laquelle beaucoup d'entre nous s'appuyaient — un relecteur gratuit qui commentait discrètement chaque PR — a disparu depuis aujourd'hui, et Antigravity n'en est pas un remplacement direct. Migrer vers Antigravity vous garde dans l'univers de Google ; adopter le relecteur de Copilot, CodeRabbit, Qodo ou Greptile vous rend vos revues de PR. Quoi qu'il en soit, aujourd'hui est un bon jour pour brancher le remplaçant avant que la prochaine pull request ne passe sans relecture.

Sources : la page des dépréciations de Gemini Code Assist de Google (dates d'arrêt et exception entreprise) et son billet sur la transition de Gemini CLI vers Antigravity CLI ; reportage de 9to5Google. Les détails sur les outils proviennent du site de chaque fournisseur en juillet 2026 ; les tarifs changent souvent, vérifiez donc avant de vous engager.
