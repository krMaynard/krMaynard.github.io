Analyse

Données de transparence

Tableaux de bord interactifs construits à partir des rapports de transparence des plateformes — réunissant en une seule vue comparative des données que les entreprises publient séparément selon les services ou les périodes. Tout le traitement se fait dans votre navigateur ; aucune donnée n'est envoyée à un serveur.

À la une · API en direct

API de requêtes de transparence DSA VLOP

Une API de recherche en direct et un tableau de bord interactif sur les données de transparence du règlement européen sur les services numériques (DSA) provenant de 33 très grandes plateformes en ligne désignées (H2 2025). Une API asynchrone (tâche/interrogation) dotée d'une interface de requêtes structurées sûre, inspirée de la TikTok Research API. Parcourez les statistiques d'application inter-plateformes, exécutez des requêtes structurées sur les 9 tables de rapport DSA, ou posez vos questions en langage naturel grâce à la boîte « Ask » propulsée par Claude.
